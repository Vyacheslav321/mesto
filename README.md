# Проектная работа №4: Mesto-Russia

Данный репозиторий создан в рамках обучения на [Yandex.Practicum](https://practicum.yandex.ru "Я практикум")

### Обзор

Работа сделана по проекту в [Figma](https://www.figma.com/file/2cn9N9jSkmxD84oJik7xL7/JavaScript.-Sprint-4?node-id=0%3A1)

**В работе использованы** CSS технологии FlexBox и Grid. Так же использовались медиа-запросы и псевдоклассы. С помощью JS выполнен функционал по вводу имени и работы через попап. В JS  использовались функции открытия/закрытия попапа и закрытие с сохранением введенных данных. Так же реализован выход из попапа по нажатию ESC. Адаптивная верстка выполнена согласно проекта для базоых разрешений 320px и 1280px. Верстка выполнена по методологии БЭМ.
